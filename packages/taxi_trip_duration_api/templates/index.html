NYC TAXI TRIP DURATION

This application predicts taxi trip durations in NYC with an AI model built from 1.5 million Historical NYC Taxi Trips, current weather data, and geographical data. The process without calculating routes is light to make predictions fast. Simply click pickup/dropoff points, set the passenger count, and press the prediction button is all it needs.







HISTORICAL TRIPS BY HOUR

3000 taxi trips randomly selected from the historical data are animated below by hour. Bubble size represents trip distance. Zoom in and hover over to see trip details.
